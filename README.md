# css-max-width-scale

Functional CSS for max-width-scale

## Filesize

| File | Size |
|------|------|
| `dist/max-width-scale.css` | 1665 bytes |
| `dist/max-width-scale.min.css` | 1111 bytes (224 Gzipped) |

## Install

```sh
npm install css-max-width-scale
```

## Usage

### Import

```css
@import "css-max-width-scale";
```

### CDN

```html
<link rel="stylesheet" href="https://unpkg.com/css-max-width-scale/dist/max-width-scale.min.css">
```

### Direct

```html
<link rel="stylesheet" href="path/to/css-max-width-scale/dist/max-width-scale.min.css">
```

### Responsive

Responsive variants are available for each class with the following suffixes:

| Suffix | Media Query              |
|--------|--------------------------|
| `-s`   | `min-width: 32em`        |
| `-m`   | `min-width: 48em`        |
| `-l`   | `min-width: 96em`        |

Example: `.mx-wi1-m` applies the property at the medium breakpoint and above.

## Building

```sh
npm run build
```

Processes `src/max-width-scale.css` with [Lightning CSS](https://lightningcss.dev) and outputs to `dist/`.

- `dist/max-width-scale.css` — formatted
- `dist/max-width-scale.min.css` — minified
